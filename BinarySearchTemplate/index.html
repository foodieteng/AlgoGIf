01核心：TF 陣列思維

把 check(mid) 的結果想成布林陣列。二分搜尋就是在找 FFFTTTT 的交界。

範例：找第一個 ≥ 5 的位置

nums = [1, 3, 5, 5, 7, 9]，check(i) = nums[i] >= 5

F0·1
F1·3
T2·5
T3·5
T4·7
T5·9
▲ 目標 = 第一個 T 的 index = 2
只要讓陣列是單調的 F…FT…T，二分就能用。這條界線就是 boundary，演算法用 log n 次逼近它。

02l < r 標準模板

最不容易寫錯的版本。終止時 l == r，同時指向 boundary。

C++ 模板：找第一個 T
int binarySearch(vector<int>& nums, auto check) {
    int l = 0, r = nums.size();        // r = n，不是 n-1
    while (l < r) {
        int mid = l + (r - l) / 2;       // 偏左，避免溢位
        if (check(mid)) r = mid;             // T：答案可能是 mid 或更左
        else             l = mid + 1;         // F：答案一定在右邊
    }
    return l;                              // 第一個 T 的位置
}
不變量
[0, l) 全 F，[r, n) 全 T
終止
l == r 就是第一個 T 的位置
無 T
回傳 n（l 一路推到 n）
全 T
回傳 0

int lowerBound(vector<int>& nums, int x) {
    int l = 0, r = nums.size();
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (nums[mid] >= x) r = mid;   // T
        else                   l = mid + 1; // F
    }
    return l;
}

int upperBound(vector<int>& nums, int x) {
    int l = 0, r = nums.size();
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (nums[mid] > x) r = mid;    // T
        else                  l = mid + 1; // F
    }
    return l;
}
記憶訣竅： lower 用 >=，找起點 upper 用 >，找終點 + 1
常用公式：
· x 的出現次數 = upper_bound(x) - lower_bound(x)
· x 最後一個位置 = upper_bound(x) - 1
· x 是否存在 = lower_bound(x) != nums.end() && *it == x

07STL：std::lower_bound / upper_bound C++

內建於 <algorithm>，用在已排序的 vector / array。回傳 iterator。

基本用法
#include <algorithm>
vector<int> v = {1, 3, 5, 5, 7, 9};

auto it1 = lower_bound(v.begin(), v.end(), 5);  // → v[2]
auto it2 = upper_bound(v.begin(), v.end(), 5);  // → v[4]

int idx = it1 - v.begin();                        // iterator → index
int cnt = upper_bound(v.begin(), v.end(), 5)
        - lower_bound(v.begin(), v.end(), 5);    // 5 出現 2 次

// 一次拿到 [lower, upper)
auto [lo, hi] = equal_range(v.begin(), v.end(), 5);
檢查是否存在： 
auto it = lower_bound(v.begin(), v.end(), x);
if (it != v.end() && *it == x) { /* 找到 */ }
自訂比較函式（降冪陣列）： 
auto it = lower_bound(v.begin(), v.end(), x, greater<int>());
// 第一個 <= x 的位置（因為比較反了）

08STL：set / multiset C++

紅黑樹，一律用成員函式。

最重要的地雷：兩種 lower_bound 的差異
寫法複雜度評價
s.lower_bound(x)O(log n)✓ 正確
std::lower_bound(s.begin(), s.end(), x)O(n)✗ 會編譯過，但退化
set iterator 不是 random access，std::lower_bound 會走成線性。
對 set / multiset / map 一律用成員版：s.lower_bound(x)。
四神獸：找最接近 x 的元素
set<int> s = {10, 20, 30, 40, 50};

// ① 最小 >= x  （後繼，含相等）
auto it = s.lower_bound(x);
if (it != s.end()) cout << *it;

// ② 最小 >  x  （嚴格後繼）
auto it = s.upper_bound(x);
if (it != s.end()) cout << *it;

// ③ 最大 <= x  （前驅，含相等）
auto it = s.upper_bound(x);
if (it != s.begin()) { --it; cout << *it; }

// ④ 最大 <  x  （嚴格前驅）
auto it = s.lower_bound(x);
if (it != s.begin()) { --it; cout << *it; }
口訣：
· 想要「≥ / >」用原版
· 想要「≤ / <」先拿「> / ≥」再 --it
multiset：erase 與計數
multiset<int> ms = {1, 2, 2, 2, 3};

ms.erase(2);                   // ❌ 刪掉全部的 2

auto it = ms.find(2);          // ✅ 只刪一個
if (it != ms.end()) ms.erase(it);

int cnt = ms.count(2);           // x 出現次數

auto [lo, hi] = ms.equal_range(2); // [lower_bound, upper_bound)
unordered_set / unordered_map 沒有 lower_bound！hash table 沒有順序。要二分一定用有序版 set / map。

09STL：map C++

依 key 排序，二分只對 key 操作。

map<int, string> m = {{1,"a"}, {3,"c"}, {5,"e"}};

auto it = m.lower_bound(3);   // 第一個 key >= 3 → {3, "c"}
auto it2 = m.upper_bound(3);  // 第一個 key >  3 → {5, "e"}

if (it != m.end())
    cout << it->first << " = " << it->second;
容器特性二分支援
set唯一、有序✓ 成員版
multiset允許重複、有序✓ 成員版
mapkey-value、依 key 有序✓ 成員版
unordered_sethash、無序✗
unordered_maphash、無序✗
vector（已排序）隨機存取✓ std:: 函式版

10實戰題型

① 動態前驅 / 後繼查詢

串流資料邊插入邊查詢「最接近 x」。

multiset<int> ms;
ms.insert(x);

auto it = ms.lower_bound(x);            // 後繼
if (it != ms.begin()) {
    auto prev = prev(it);               // 前驅
}
② LIS 最長遞增子序列 O(n log n)
vector<int> tails;
for (int x : nums) {
    auto it = lower_bound(tails.begin(), tails.end(), x);
    if (it == tails.end()) tails.push_back(x);
    else                    *it = x;
}
return tails.size();
嚴格遞增 → lower_bound；允許相等 → upper_bound
③ 找離 x 最近的元素
auto hi = s.lower_bound(x);
int best = INT_MAX;
if (hi != s.end())   best = min(best, *hi - x);
if (hi != s.begin()) best = min(best, x - *prev(hi));
④ 離散化
vector<int> v(nums.begin(), nums.end());
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());

// 原值 x 離散化後的 index
int idx = lower_bound(v.begin(), v.end(), x) - v.begin();
⑤ 二分答案（最小可行 k）

Koko 吃香蕉、切繩子、運送包裹這類題目：對「答案」做二分。

int l = 1, r = 1e9;
while (l < r) {
    int mid = l + (r - l) / 2;
    if (feasible(mid)) r = mid;      // 能做到，試更小
    else                 l = mid + 1;
}
return l;

寫題 SOP
  1. 畫 TF 陣列：根據題目條件畫出 F…FT…T 或 T…TF…F。
  2. 定義 check(mid)：讓答案 = 第一個 T。
  3. 選模板：偏左 mid + r = mid + l = mid+1。
  4. 決定回傳：第一個 T → l；最後一個 F → l - 1。
  5. 處理邊界：無 T → l = n；全 T → l = 0。
